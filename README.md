# valderas.be

Personal portfolio and blog — built with vanilla HTML, CSS, and JavaScript.

## Setup

1. Clone this repo
2. Enable GitHub Pages in **Settings → Pages → Source: main branch**
3. (Optional) Add a custom domain: `valderas.be`

## Structure

```
├── index.html              # Homepage
├── portfolio/              # Case study pages
│   ├── vrt-redesign.html
│   ├── osoc.html
│   ├── interactive-strategy.html
│   └── conversational-ai.html
├── blog/                   # Blog posts
│   ├── designing-with-ai.html
│   ├── open-source-design.html
│   └── micro-interactions.html
├── assets/
│   ├── css/style.css       # All styles
│   ├── js/main.js          # Micro-interactions & interactivity
│   └── img/                # Images
└── README.md
```

## Features

- Clean black & white design
- Custom cursor with follower
- Magnetic button effects
- Scroll-triggered reveal animations
- Responsive mobile navigation
- Subtle parallax on hero
- Label system with mono-spaced typography
- No build tools or dependencies — pure HTML/CSS/JS

## Custom Domain

To use `valderas.be`:
1. Add a `CNAME` file with `valderas.be`
2. Configure DNS: A records pointing to GitHub Pages IPs
3. Enable HTTPS in repo settings
